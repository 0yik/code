POS Multiple Receipts

POS Multiple Receipts is a module to print multiple receipts of an order in POS Session. By default in POS, only one receipt will be printed for an order. With the help of this module, seller can print more than one receipt for an order. This module is useful in such a scenario:-

  • Scenario 1- When seller wants to print one receipt for customer and other one for his purpose.
  • Scenario 2- When seller wants to print receipts from multiple printers located at different locations.

Features
  • Seller can print more than one receipt from main printer.
  • Seller can add multiple printers located at different location to print multiple receipts.
  • Seller can print original receipt as well as duplicate receipts.
  • Display the status of all additional printers (connected or disconnected) in running session.
  • No complex backend configuration
  • POS Session View
  • Backend Configuration
