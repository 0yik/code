Odoo 10 Point of Sale Extension

This module allow to configure POS cross selling products and use them during sales.


Note: This module is tested and working in community edition.

How to use

  • Configure cross selling products with price after applying discount.


  • During adding of product from POS screen, open popup with optional products.


  • Add selected products into POS cart.


  • Same products are display into POS Receipt.


  • It is keeping history of cross selling products.
