POS Warehouse Management

POS Warehouse Management is a module to manage multiple warehouse within POS session. It shows and manage products stock in POS session. By default in POS only one stock location can be selected at one time. This module allow the POS user to select more than one stock locations in POS, So that if any product is out of stock then POS user can order product from other stock locations.

Features
  • Display the current stock of products in running session.
  • Seller can select multiple stock locations for a single Point Of Sale by setting the other related stock locations
  • If product is out-of-stock in main stock location, seller can add product from other related stock locations.
  • Stock will be updated automatically after order validation
  • POS Session View
  • Backend Configuration
