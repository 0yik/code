Import Serial/Lot number with stock Inventory by CSV or XLS file

This module provides the feature which will import stock inventory adjustment using CSV or XLS file with or without serial number.

key features:
  • Easy to import opening stock balance from Excel or CSV file with serial number.
  • Generate inventory adjustment for opening stock.
  • Import product line with or without serial number.
  • If same serial number exist with same product(on CSV) link that serial number imported on inventory line . If not then it will create new serial number for that product and import with its stock.

Given below is the sample of import file,It is mandatory to use given format to create CSV or XLS file.



Code
- Code of the product for which you want to create inventory
Quantity
- Quantity of particular product .
UOM
- Unit of Measure of product .
Lot/Serial Number
- Unique Lot/Serial Number.

Product Lot/Serial Number : If you have entered the Lot number which is not available in the system, then it will assign new Lot/Serial number for that product.


Unavailable UOM Error: If you have entered the UOM which is not available in the system, then following error will appear.
