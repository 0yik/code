Odoo 10 Sales vs Cash Report

Customize Sales vs Cash Report


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • You can see the available Custom Reports Menu for Sales vs Cash Report.


  • The Sale vs Cash Reports opens a wizard to select the date.


  • Xls will show last 90 days record how much you sale in cash and credit for all the partners.
