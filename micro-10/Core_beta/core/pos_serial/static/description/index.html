Odoo 10 Point of Sale Extension

Assign lot/serial number(s) from Point of Sale Front-end

Let's see the video

Note: This module is tested and working in community edition.

Features

  • Search particular serial/lot number(s).
  • Assign more than one quantity of single serial/lot number(s) at a time.
  • Restrict serial product using product expiry days.

How to use

  • Configuration for enable custom serial feature.

  • List of available serial/lots.



  • We can restrict serial product expiry date wise. for example we have set product expiry days 3 so we can restrict product expire before 3 days.

  • Ability to remove assigned serial/lot number(s).


  • Printed Serial numbers on receipt.


  • Remaining serial/lot number(s).
