Odoo Product Approval

Uses :

  • 1) Default Product is create in 'Draft' state.
    2) 'Product Manager' can review and confirm. After confirmation of product user can't update product information.
    3) 'Product Manager' can also move confirm product in draft state.
    4) User only do the operation on confirmed products. For an example in sales order, purchase order and manufacturing etc..


  • Normal User that don't have Product Manager access


  • Normal user can create or update product in draft state.


  • User with Product Manager Access


  • Manager can Approve product which are in Draft state also can be Reset to Draft products which are in 'Confirmed' state.


  • User can only create sale order of confirmed products.


  • User can only create Purchase Order of confirmed products.
