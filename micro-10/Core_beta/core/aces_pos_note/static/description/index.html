Odoo 10 Point of Sale Extension

Allow user to add note for product or POS order from POS interface


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • User needs to select product for which they want to add notes or they can add notes for POS Order also.
  • We have two features available in POS Configuration as per below image.


  • After adding product we can add note for this product by clicking `Add Note` button


  • If we want to add note for POS Order so, this can be done from payment screen


  • POS Order note is printed on receipt also


  • From the back end, we can check the note for single product.


  • We can check note for POS Order also.
